## Biome Ambient Text (Loading Screen / Tooltip Samples)

**The Calibration Fields**
The first arena zone. Flat. Well-lit. The Architects' notes describe it as a "controlled introduction environment." They expected Subjects to take longer to adapt.

**The Residue Layer**
Every Subject that ever Stabilized left something behind. The Arena did not clean it up. Walk carefully — what you step on is not debris.

**The Deep Register**
No Architect log references this zone by name. The designation was found written in a format predating the Arena's own file system. Someone was here before the Architects were.

## Writing Rules for Future Boss Logs

1. Every log is a fragment — estimated integrity always < 70%
2. Written as clinical observation, never emotional
3. Always ends mid-sentence or with an unanswered question
4. Each log reveals ONE thing about the boss + ONE thing about the Arena
5. Logs reference other Subjects by designation (creates interconnected lore web)
6. Never resolve the Architect extinction mystery — only deepen it
7. Max 5 sentences per fragment
